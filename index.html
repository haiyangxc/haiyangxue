Cryptography Algorithms

LAC: Lattice-based cryptosystems, Round 2 candidate of NIST Post-Quantum Cryptography Standardization.
LAC includes Public Key Encryption, Key Encapsulation and Authenticated Key Exchange. It is based on the hardness of the RLWE problem.
[ eprint ], [Implications V1, V2], [ Round 1 submission], [ Round 2 submission], [ NIST report].
Xianhui Lu, Yamin Liu, Dingding Jia, Haiyang Xue, Jingnan He, Zhenfei Zhang, Zhe Liu, Hao Yang, Bao Li, Kunpeng Wang

SIAKE: Autheticated key exchange based on supersingular isogenies. Its security holds in both classical and quantum random oracle model.
Parameters are chosen based on SIKE .
[ eprint ], [ Document (in Chinese) ], [ Implications]
Haiyang Xue, Xianhui Lu, Kunpeng Wang, Song Tian, Xiu Xu, Jingnan He
